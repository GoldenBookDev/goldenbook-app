# Goldenbook

Goldenbook is a mobile application developed in React Native that allows users to explore and rate tourist locations. The app is designed to be intuitive and user-friendly, providing a platform for discovering new places and sharing experiences.

## Screenshots

![Alt](/goldenbook-app/assets/images/Phone-Goldenbook-Mockups%201.jpg "Phone Goldenbook Mockups")

## Technologies Used

- React Native: 18.2.0
- Expo: ~51.0.21
- TypeScript

## Main Dependencies

- React Native Elements: 4.0.0-rc.8

## Features

- **Explore Places**: Users can browse a list of nearby tourist spots.
- **Reviews and Ratings**: Ability to leave reviews and ratings to share experiences with others.
- **User Authentication**: Implementation of login and registration screens to manage user profiles.

## Installation

To install and run Goldenbook on your local environment, follow these steps:

1. Clone this repository:

```bash
git clone https://github.com/GoldenBookDev/goldenbook-app.git
```

2.Install dependencies:

```bash
cd app-goldenbook
npm install
```

Start the application:

```bash
npm start
```

This will open Expo CLI where you can choose to run the app on an emulator/simulator or on a physical device.
